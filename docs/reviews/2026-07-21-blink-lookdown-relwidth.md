# 눈감음 오탐 실측 — "아래 쳐다봄"과 상대 크기 판별축 (2026-07-21)

group 35 / event 99 사용자 리포트: 수박옷 아이가 눈을 감은 게 아니라 컵·식물을 **내려다보는데**
eyes_closed로 분류돼 인물 앨범에서 빠졌다. blink blendshape(ADR 021)의 알려진 의미론 한계
("윗눈꺼풀이 내려온 정도"를 잼)와 겹치는 축이라, threshold로 잡히는지 / 다른 축이 필요한지
전 이벤트 961장 실측으로 판정했다.

## 1. threshold로는 못 가른다 — 오탐이 정탐 분포 한가운데 박힘

event 99의 현재 flagged(eyes_closed) 얼굴을 프로덕션 경로(YuNet→blendshape)로 재측정:

| 얼굴 | blink | 실제 | |
|---|---|---|---|
| 수면 (문서 A/B) | 0.420 | 감음 | 정탐 |
| 안경 남성 (문서 A/B) | 0.465 | 감음 | 정탐 |
| 22a3d7f8 함성 질끈 | 0.516 | 감음 | **정탐** |
| 66f8c71e 식물 내려봄 | 0.530 | 내려뜸 | **오탐** |
| 기도 (문서 A/B) | 0.588 | 감음 | 정탐 |
| 1c321ea2 수박옷 컵 내려봄 | 0.608 | 내려뜸 | **오탐** |
| event 61 진짜 감음 | 0.625 | 감음 | 정탐 |
| 04bebbd5 교실 뒤 | 0.625 | 내려뜸/소형 | **오탐** |
| 71eea0d1 식물 내려봄 | 0.627 | 내려뜸 | **오탐** |
| daf7f144 먹으며 감음 | 0.637 | 감음 | 정탐 |

오탐(0.530~0.627)이 정탐(0.42~0.64) **한가운데**에 있고, event 61 정탐과 event 99 오탐이
소수점 셋째 자리까지 같은 **0.625**로 충돌. threshold를 0.63 위로 올리면 정탐 대부분이 미탐.
blendshape의 eyeLookDown 축도 진짜 감음이 함께 떠서(감음 0.61~0.89 vs 오탐 0.40~0.84) 기각.

## 2. 절대 px도 못 가른다 — 상대 크기는 갈린다

| 얼굴 | 짧은 변(px) | img 긴 변 | **rel%** | 실제 |
|---|---|---|---|---|
| 04bebbd5 교실 뒤 | 46 | 1280 | **3.6%** | 오탐 |
| 71eea0d1 식물 내려봄 | 188 | **4032** | **4.7%** | 오탐 |
| 1c321ea2 수박옷 | 58 | 1125 | **5.2%** | 오탐 |
| 66f8c71e 식물 내려봄 | 210 | **4032** | **5.2%** | 오탐 |
| 22a3d7f8 함성 질끈 | 182 | 1280 | **14.2%** | 정탐 |
| daf7f144 먹으며 감음 | 569 | 3264 | **17.4%** | 정탐 |

- **절대 px**: 정탐 182가 오탐 58·188 **사이에 끼어 중첩**. 크기로 못 가른다(64px 게이트는
  46·58만 잡고 식물 188·210은 못 잡음 — 4개 중 2개).
- **상대 크기(rel% = 폭/긴 변)**: 오탐 전부 ≤5.2% vs 정탐 전부 ≥14.2%.

원인: 식물 사진은 3024×4032 고해상도라 아이 얼굴이 절대 188~210px여도 프레임에선 4.7~5.2%
**작은 피사체**(멀리 찍힘). 함성/먹는 감은 얼굴은 14~17% **가까운 주 피사체**. 내려뜸 오탐은
멀리 찍힌 작은 얼굴, 진짜 감음은 가까운 주 피사체 — 이 상관이 rel%를 가른다. ADR-013(배경 인물
rel_w 필터)·`eye_main_face_ratio`(주 인물만 판정)와 같은 "주 피사체 vs 배경" 원리다.

## 3. 전 이벤트 961장 분포 — 빈 구간 [5.2%, 11.3%]

현재 flagged 얼굴 76개(미러 이벤트 중복 포함)를 rel%로 정렬해 육안 분류(`scripts/survey_eye_rel_width.py`):

| rel% 구간 | 대표 | 분류 |
|---|---|---|
| 2.5 ~ 5.2% | 식물 188·210px, 수박옷 58px, 교실 46px, 소형 21~32px | **내려뜸/소형 오탐** |
| 7.6% | 선글라스 여성 434px | 가림 오탐 (별개 유형) |
| 8.7 ~ 10.8% | 손 가림 93px, 안경+모션 115px | 가림/모션 오탐 (별개 유형) |
| 11.3% | 기도 여성 643px (감음) | **정탐** |
| 13.3 ~ 17.4% | 수면 538·571px, 함성 182px, 먹음 569px | **정탐** |
| 22.3 ~ 37.8% | 정면 감음·스톡 103~210px | **정탐** |

- **내려뜸/소형 오탐(리포트 대상)은 전부 ≤5.2%**, **진짜 감음은 전부 ≥11.3%**. 그 사이는
  가림·모션 오탐(다른 실패 유형)뿐 — 정탐 0.
- 빈 구간 [5.2%, 11.3%]에서 **0.08(8%)** 채택: 오탐 상한 5.2% 대비 1.54배, 정탐 하한 11.3%
  대비 1.41배 마진. 검출 rel 하한(2.5%)은 앨범 편입용이라 더 낮다 — 얼굴은 넣되 눈감음만 더
  엄격히 요구.

## 4. 회귀 (전 이벤트 959장 before/after diff)

`eye_min_rel_width` 0 vs 0.08 이미지 단위 eyes_closed 비교(`verify_rel_gate.py`):
- 판정 변경 **37장, 전부 True→False(해제), False→True 0**(게이트 단조성 확인).
- 해제 = 내려뜸/소형 오탐 4종 × 미러 이벤트 8개(32) + 소형 얼굴(event 5·13·89, 4) + 선글라스
  가림 오탐(event 83, 1). **진짜 감음 이미지(≥11.3%)는 한 장도 안 뒤집힘 — 미탐 0.**

## 5. 한계 (게이트가 못 잡는 것)

- **프레임에서 큰 내려뜸**: event 103 아이(웃으며 내려봄, rel **32.8%**)는 게이트 위라 잔존 —
  이 축으로 못 거른다. 웃음 실눈 캔디드와 같은 의미론 문제.
- **가림 오탐**(선글라스 7.6%·모션 10.8%)은 이 게이트 대상 아님 — ADR-019 계열의 별도 문제.
- **원거리 진짜 감음**은 미판정(주 피사체 아님 — ADR-013 원리상 허용, 사용자 트레이드오프 확인).

도구: `scripts/survey_eye_rel_width.py`(전 이벤트 flagged 얼굴 rel% 수집·크롭).
