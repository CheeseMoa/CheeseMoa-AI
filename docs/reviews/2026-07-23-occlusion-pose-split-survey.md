# 가림·자세(선글라스·모자·옆모습) 갈라짐 실측 — 조건별 유사도 + 실코퍼스 yaw 상관

- 날짜: 2026-07-23
- 도구: `scripts/survey_occlusion_pose.py`(셀럽 조건 세트) · `scripts/survey_yaw_corpus.py`(실코퍼스
  46 이벤트) · `scripts/survey_sunglasses_blendshape.py`(선글라스 신호 후보) — 전부 재실행 가능
- 데이터: `image/images/conditions/`(신규 수집 24장, 출처·검증은 그 README) + `downloaded/` 기준선 +
  S3 실코퍼스(재검출 900 얼굴, 내용 중복 제거 후 고유 455)

## 문제

같은 사람이 선글라스·모자·옆모습 사진에서 다른 인물로 갈라진다는 체감 리포트. 어떤 조건이 실제로
얼마나 유사도를 무너뜨리는지, 값싼 감지 신호가 있는지 실측했다.

## ① 조건별 유사도 — 병합 게이트(centroid 0.55 AND face-pair 0.475) 대비

측정 대상 얼굴은 전수 육안 검증(`scripts/occlusion_survey/targets_*.jpg`). 최대 bbox 선택이 동반
인물을 집는 오염 2건을 발견해 "단일 얼굴 사진 기준벡터와 가장 유사한 얼굴 채택"으로 교정했다.

| 조건 | 게이트 | 수치 (centroid / face-pair 평균) |
|------|--------|-------------------------------|
| 모자 단독 | **통과** | hamilton 0.765/0.581 · woods 0.606/0.484 |
| 모자+선글라스 | 탈락 | hamilton 0.609/0.470 |
| 선글라스 | 탈락 | psy 0.591/**0.334** (개별 쌍 최저 0.133) |
| 옆모습 | 갈림 | iu 0.440·merkel 0.447 탈락 vs obama 0.699/0.562 통과 |
| 마스크 | 통과 | moon 0.581/0.524 (n=1) |

- **모자 자체는 무해** — "모자 때문"이라는 체감의 진범은 모자+선글라스 조합·챙 그림자·고개 각도일
  가능성. 실서비스 사진으로 재분해 필요.
- **선글라스는 face-pair에서 죽는다** — centroid는 턱걸이 통과라 파편 병합의 1차 관문이 아니라
  2차 관문(ADR 016)에서 갈라진다.
- **조건 내 응집 > 조건↔정면**: obama 옆모습끼리 0.705(정면과 0.562), hamilton 모자선글라스끼리
  0.685(정면과 0.470) — 조건별 프로토타입(다중 대표) 매칭의 근거. 단 psy 선글라스끼리는 0.356
  (안경 모양·조명이 제각각이면 응집 안 됨) — 만능 아님.
- **임계 완화 금지 근거**: 남남 정면 최대가 iu↔suzy 0.482로 face-pair 게이트(0.475)를 이미 넘는다.
  같은 조건 남남은 이 표본에선 낮았으나(최대 0.108) 표본이 작고 인구통계가 갈린 쌍들뿐이라 유보.
- 부산물: taylor 기준선 자체가 0.384로 게이트 미달 — 17년 나이 변화는 가림과 별개의 갈라짐 원인.

## ② 실코퍼스 — yaw 근사와 군집 결과의 상관

5점 랜드마크 yaw 근사: 코끝의 눈중점 대비 오프셋을 눈축 단위벡터에 투영, 눈간격으로 정규화(롤 불변).
완전 측면에선 눈간격(분모)이 붕괴해 값이 폭주(최대 6.8)하지만 플래그 용도로는 단조라 무해.
S3 46 이벤트를 재검출해 저장 bbox와 IoU 매칭(900 얼굴, 불일치 4). 테스트 이벤트들이 같은 사진
세트를 공유해 crop 내용 md5로 중복 제거한 고유 455 얼굴 기준(다수결 결과):

| yaw 구간 | n | 미배정율 |
|----------|----|---------|
| [0.00, 0.30) | 342 | 35.7% |
| [0.30, 0.45) | 32 | 50.0% |
| [0.45, 0.60) | 21 | 76.2% |
| [0.60, 0.80) | 17 | 82.4% |
| [0.80, ∞) | 43 | **90.7%** |

임계 후보(고유 얼굴 기준): **T=0.50 → 플래그 72개, 정밀도 88.9%, 전체 미배정의 30.9% 커버,
배정 얼굴 오플래그 3.2%.** T=0.45는 커버 33.3%/오플래그 4.8%. 몽타주 육안 검증
(`scripts/yaw_corpus/montage_*.jpg`): 상위 60개 전원 진짜 완전 측면(오탐 0), 회색지대(0.40~0.60)도
대부분 실제 반측면·기울어짐.

**설계 제언**: yaw 플래그는 **후처리 원인 라벨**로 쓰는 게 안전하다 — 재군집이 끝난 뒤 미배정
얼굴 중 yaw ≥ T에 uncertain `causes`로 "옆모습"을 다는 방식(CHMO-404 계약의 자연 확장). 이러면
오플래그 3.2%는 이미 배정된 얼굴이라 영향이 없다. 반대로 선제 배제(임베딩·군집 제외)는 멀쩡히
배정될 3.2%를 잃으므로 신중해야 한다. 단독 신규 군집 생성 차단(유령 앨범 방어)에 쓰는 것은 중간
강도의 절충. 부수 발견: 같은 얼굴이 이벤트에 따라 배정↔미배정이 갈리는 사례 8건(경계선 얼굴의
이벤트 구성 의존성).

## ③ 선글라스 감지 신호 — blendshape 기각

Face Landmarker(blink, ADR 021)의 presence·blink 출력은 선글라스에서 정면 민낯과 구분 불가
(**선글라스 전건 presence 1.000**, minBlink 분포도 동일 — 랜드마커가 렌즈 뒤 눈을 "뜬 눈"으로
환각). 눈감음 오판은 0건이라 품질 게이트에는 무해. **선글라스 감지는 별도 신호가 필요하다**(눈
영역 crop의 색·에지 통계, 소형 분류기 등 — 후속 실측 대상).

부수 발견: **옆모습에서 presence가 붕괴**(평균 0.562, 최저 0.000, n=4) — yaw 근사를 보완할 옆모습
보조 신호 후보.

  README §커버리지 공백). 팀 동의 사진 보강 필요.

  31%는 "미배정의 1/3이 자세 문제"라는 뜻이지 나머지가 정상이라는 뜻이 아니다.
- v2 시절 .npz 행 278개는 s3_key가 없어 재검출 매칭에서 제외됐다.

## 결론 (방법 후보 대응)

1. yaw 근사 → uncertain `causes` 후처리 라벨: **근거 확보, 구현 후보 1순위** (T≈0.5, 추가 모델 0).
2. 조건별 프로토타입 병합: 옆모습·동일 안경류에는 근거 있음, 선글라스 일반화는 안 됨 — 병합
   게이트 재보정 리스크 있어 별도 실험 필요.
3. 병합 임계 완화: **기각** (iu↔suzy 0.482가 게이트 위).
4. blendshape 선글라스 감지: **기각**. 별도 신호 실측으로 대체.
5. 모자 단독 대응: **불요** (게이트 통과). 체감 리포트는 조합·그림자 케이스로 재분해할 것.
