# ADR 020: 저신뢰 분리 회색지대 face-pair 재확인 — 남남 부착 축출

## Status

Accepted (2026-07-17). 재군집 후처리 ⑤ 저신뢰 분리(`_evict_ambiguous`)에 회색지대 게이트 추가.
ADR-016(파편병합 face-pair 응집 게이트)과 같은 원리를 축출 방향에 적용한다.

## Context

group 27 / event 61에서 남남 여성 얼굴(눈감음 CNN 미탐으로 파이프라인에 진입한 보정 스톡 이미지)이
6인 클러스터(인물 55)에 남았다. 실측: 이 얼굴의 클러스터 전 쌍 유사도 0.279~0.393(전부 타인 상한
0.4584 미만, 내부 쌍의 53%가 타인 수준)인데, leave-one-out centroid 유사도가 **0.425**로 저신뢰
분리 바닥(`min_membership_similarity=0.4`)을 0.025 차로 통과했다.

### 전역 바닥 상향 스윕 — 기각 (2026-07-17, `scripts/tune_membership_floor.py`)

바닥을 0.42~0.48로 올리는 스윕(라벨 코퍼스 5폴더 + S3 이벤트 56개):
- event 61을 고치는 최소값은 0.43인데, **0.42에서 이미 similar2(단일인물) frac 1.0→0.8**,
  0.43에서 similar3 1.0→0.75, 0.46에서 similar 0.833→0.667로 진짜 멤버가 잘린다.
- 불변식(`blob_promote_floor ≥ min_membership_similarity`) 때문에 바닥 상향은 blob 승격 바닥을
  함께 끌어올려 승격으로 형성되던 단일인물 클러스터도 깨진다 (코퍼스 회귀의 주 경로).
- 실 이벤트에서도 0.44부터 동일인 증거(top쌍 0.554 ≥ 병합 임계)를 가진 얼굴 3건이 강등된다.
- 근본 원인: **동일인 LOO와 남남 LOO가 [0.40, 0.46)에서 섞여** 전역 임계로는 원리적으로 분리 불가.

### 판별 신호는 개별 쌍에 남는다 (사분면 실측)

강등 후보(LOO < 0.46) 전수의 "클러스터 내 최강 face-pair":

| 코호트 | LOO centroid | top face-pair |
|---|---|---|
| 남남 부착 (실 이벤트 20건, event 61 여성 포함) | 0.402~0.456 | **≤ 0.440** |
| 동일인 증거 보유 (실 이벤트 3건) | 0.435 | **≥ 0.554** |
| 라벨 코퍼스 진짜 멤버 (전원) | ≥ 0.502 | ≥ 0.469 |

centroid가 못 가르는 것을 top쌍이 빈 구간 [0.440, 0.469]를 두고 가른다 — ADR-016이 병합에서
확인한 것("아동 centroid는 평균으로 뭉개진다, 판별 신호는 쌍에 남는다")과 동일 구조.

## Decision

`_evict_ambiguous`에 회색지대 face-pair 재확인을 추가한다: LOO centroid가 바닥(0.4)은 넘었지만
`evict_gray_ceiling`(기본 **0.46**) 미만인 멤버는, 클러스터 내 최강 face-pair가
`evict_facepair_floor`(기본 **0.45**) 미만이면 — 클러스터의 누구와도 동일인 증거가 없는 남남
부착이므로 — 노이즈로 축출한다(uncertain행). 둘 중 하나라도 0이면 비활성.

값의 근거:
- ceiling 0.46 = 남남 부착 관측 최고 LOO 0.456 직상, 코퍼스 진짜 멤버 최저 LOO 0.502 아래.
- facepair floor 0.45 = 실측 빈 구간 [0.440, 0.469] 안이면서 **blob 승격 간선 임계(0.45)와 동일값**
  — 승격 성분의 전 간선이 floor를 자동 충족해, 방금 승격된 blob이 같은 실행의 이 게이트로
  해체되는 churn을 구조적으로 차단한다. `__post_init__` 불변식 2개 추가
  (ceiling ≥ membership 바닥, facepair floor ≤ blob 승격 간선).

## Consequences

재검측 (라벨 코퍼스 + S3 이벤트 45개(2얼굴 이상), 게이트 OFF 대비 전수 비교):
- **라벨 코퍼스 회귀 0** — child ARI 0.784 / different 1.0 / 단일인물 frac 전부 불변.
- 실 이벤트: 13개 이벤트에서 20얼굴 강등, **전부 top쌍 < 0.4584(타인 정합)**, 동일인 증거 보유
  0건, 의도 밖 변화(승격·이동) 0건. 나머지 32개 이벤트 완전 불변.
- **event 61 해소**: 남남 여성이 인물 55에서 uncertain으로 이동 (top쌍 0.393).
- 강등 20건의 의미: 남남 "확정"이 아니라 "동일인 증거 없음"이다 — 극한 가림·저품질로 신호가
  없는 진짜 멤버도 포함될 수 있으나, 그런 얼굴은 인물 앨범이 아니라 uncertain이 맞다는 것이
  기존 저신뢰 분리 정책(feature-spec §7)이고, 사용자 보정(reassign)으로 복구 가능하다.
- 한계: 게이트는 회색지대(LOO < 0.46)만 본다 — LOO ≥ 0.46으로 강하게 부착된 남남(관측 사례
  없음)은 못 잡는다. 도구: `scripts/tune_membership_floor.py`(스윕 + 강등 판별).
