# ADR 024: 파편병합 승인을 컴포넌트 전체 재평가로 — 스냅샷 완전 연결의 소형 파편 노이즈 해소

## Status

Accepted (2026-07-20). `cluster.py` `_merge_fragments`의 병합 승인 검사를 "병합 전 파편 스냅샷 쌍의
완전 연결(complete linkage)"에서 "두 컴포넌트의 **현재 전체 멤버** 재평가"로 교체 — 재계산 centroid
≥ `merge_centroid_similarity`(0.55) AND 전체 얼굴 교차 face-pair 평균 ≥ `merge_facepair_floor`(0.475).
새 임계값 없음(기존 게이트 값 재사용). `CLUSTER_MERGE_COMPONENT_LINKAGE`, false = 구 동작 롤백.

## Context

group 35 / event 90 실 사용자 리포트: 같은 인물인데 앨범이 갈라져 있다. 진단
(`scripts/diagnose_event.py` + 병합 계측)으로 층을 가른 결과, 이벤트는 인물 3명인데 주 인물의
2얼굴 파편 앨범 2개가 별도 앨범으로 남았고, 그중 하나(`ff24563e`)는 **병합 게이트를 통과하고도**
못 붙고 있었다:

1. 2차 병합(ADR-010) 시점의 파편은 4개 — 주 컴포넌트(13), 소형 파편 3개(각 2얼굴).
2. 소형 파편 X는 주 컴포넌트와 centroid 0.669/face-pair 0.526으로 병합(정상).
3. 소형 파편 Y(`ff24563e`)도 주 컴포넌트와 0.632/0.479로 게이트 통과. 그러나 완전 연결 검사가
   "이미 합류한 X와 Y의 **병합 전 스냅샷 쌍**"까지 게이트를 요구 — X·Y는 같은 인물의 파편인데
   2얼굴짜리 centroid의 표본 노이즈로 0.508/0.458에 그쳐 미달 → Y 합류 차단, 별도 앨범 잔존.

핵심 결함: 완전 연결의 단위가 "병합 전 파편 스냅샷"이라, 소형(2얼굴) 파편이 하나 합류할 때마다
이후의 모든 후보가 그 노이즈 낀 스냅샷과도 직접 닮을 것을 요구받는다. 15얼굴 컴포넌트가 주는
안정된 증거(0.641/0.476)는 반영되지 않는다. 파편이 작을수록 — 즉 증거가 가장 필요한 곳에서 —
검사가 가장 부정확해지는 구조다.

임계값 조정으로는 해결 불가함을 먼저 실측했다(2D 그리드 스윕, 라벨 코퍼스 + 이벤트 52개):
Y를 붙이는 어느 (centroid, floor) 조합도 ADR-016 재보정이 잡았던 아동 이벤트 오병합을 되살린다
— floor 0.45에서 이벤트 7개(38·43·54·56·60 등) 최대 앨범 내부 타인쌍 6%→37~39%, floor 0.42에서
15개로 확산 + child ARI 0.784→0.610. 임계의 문제가 아니라 검사 구조의 문제다.

## Decision

greedy 병합 루프의 승인 검사를 컴포넌트 전체 재평가로 교체한다 (`merge_component_linkage=True` 기본):

- 후보 생성(파편 쌍 게이트 통과 쌍)·greedy 순서(스냅샷 유사도 내림차순)·cannot-link 차단은 불변.
- 승인 시점에 두 컴포넌트의 **현재 멤버 전체**로 재계산한 centroid 유사도와 전체 얼굴 교차
  face-pair 평균에 **같은 임계**(0.55 / 0.475)를 적용한다.
- 완전 연결이 막던 다리(bridge) 융합 — 닮은 중간자 B가 남남 A·C를 잇는 전이 체인 — 은 전체
  face-pair 평균이 대행한다: 컴포넌트에 남남 얼굴이 섞여 있으면 남남 쌍(≲0.3)이 평균을 바닥
  아래로 끌어내린다(합성 기하 실측 0.44 < 0.475, 자가검증 (m)).

1차·2차 병합(ADR-010) 공통. event 90에서 Y는 주 컴포넌트(15얼굴 전체) 대비 0.641/0.476으로
자연 통과해 합류한다.

  같은 사진 cannot-link, ADR-016 face floor 포함) 구·신 양쪽 전부 통과. 신규 (m) 2종 추가 —
  event 90 기하(구 방식 분리 재현 → 신 방식 치유), 다리 기하(신 방식도 융합 차단).
- **라벨 코퍼스 5폴더**: child ARI 0.784 / different 1.000 / 단일인물 frac 3종 — 구·신 완전 동일.
- **실 이벤트 52개**(현 S3 8 + 과거 캐시 44, 아동 오병합 이벤트 35·38·43·54·56·60 포함): 변화는
  event 90 단 하나 — [15, 9, 8, 2, 2] → [17, 9, 8, 2] (파편 Y 합류, 의도된 치유). 나머지 51개
  파티션 불변.

## 한계·리스크

- **거대 다리 컴포넌트**: 한 인물 컴포넌트가 충분히 크면(예: 15얼굴) 남남 소형 파편이 섞인
  뒤에도 물량이 전체 face-pair 평균을 바닥 위로 유지시켜, 구 방식이라면 막혔을 남남 합류가
  이론상 가능하다. 단 "한 컴포넌트가 서로 남남인 두 인물 모두와 0.55/0.475를 넘는" 전제 자체가
  사실상 아동 영역인데, 그 영역(child 코퍼스 + 아동 이벤트 6개)이 전부 무변화로 실측됐다.
  아동 오병합 리포트가 오면 구성 파편 쌍에 남남 수준 하한(veto)을 추가하는 방향으로 재보정한다
  — 단 event 90의 정상 쌍이 0.458이므로 타인 상한 0.4584(ADR-012)를 그대로 veto로 쓰면 안 된다.
- **다른 소형 파편(`18c6dd40`)은 못 붙인다(의도)**: 주 앨범과 face-pair 0.420, 얼굴별 최근접
  0.533~0.551로 동일인 증거 자체가 없다(임베딩 품질 문제). 병합 로직이 아니라 사용자 merge
  보정(must-link) 영역이며, 억지로 붙이는 임계는 위 스윕대로 아동 오병합을 되살린다.
